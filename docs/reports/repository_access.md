# Repository Access Instructions

- Remote URL: https://github.com/modalgrasp/AI_F1_Driver.git
- Default branch: main

## Clone
```bash
git clone https://github.com/modalgrasp/AI_F1_Driver.git
cd F1
git checkout main
```

## SSH Setup
```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
# add public key to your git provider
```

## HTTPS Token Setup
Create a personal access token with repository scope and use it as password when prompted.
